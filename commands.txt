Инициализация Git-репозитория: git init
Для того что бы отслеживать файлы или добавить их под версионный контроль используется:git add<>
Фиксация изменений (коммит): git commit -m "text"
Просмотр истории коммитов: git log
Отправка изменений в удаленный репозиторий: git push origin <branch>
отображения состояния рабочей директории и индекса: git status
